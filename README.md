## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/panteliza/Clouds-Nepal-Web.git
```

Navigate to the project folder:

```bash
cd Clouds-Nepal-Web
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
